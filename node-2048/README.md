# 2048
Popular game recreate in javascript

### Installation
1. Clone the repo 
```sh 
git clone https://github.com/melsen888/GAME
```
or
Download file
2. Install NPM packages
```sh
npm install
```

## Usage

> ```sh
> npm run start
> ```
